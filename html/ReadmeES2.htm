Acerca de este controlador

Este es un controlador para utilizar D-05 en macOS 11.

Características

  • Compatible con los estándares Apple de CoreAudio y CoreMIDI
  • Puede personalizar el tamaño del búfer del controlador y gestionar latencias precisas de grabación y reproducción

Requisitos del sistema

Este controlador tiene los siguientes requisitos.

*Compruebe también los Requisitos del sistema para la aplicación MIDI/de audio y el sistema operativo que utilice.
Sistema operativo
macOS 11
 
Ordenador
Ordenador Apple Mac con USB integrado
*Aunque se conecte a un puerto USB 3.0, el rendimiento del producto no cambiará.
*Los equipos Mac con Microsoft Windows no son compatibles.

Precauciones y limitaciones

  • Antes de iniciar el secuenciador o el software relacionado, utilice un cable USB para conectar el D-05 al ordenador y encienda el D-05.
  • No apague el D-05 ni desconecte el cable USB durante la reproducción o la grabación. Si lo hace, la aplicación de software o el sistema operativo podría cerrarse de forma anómala.
  • Debe salir primero del secuenciador o software relacionado y apagar después el D-05 o desconectar el cable USB.
  • Si inicia o reinicia su Mac mientras el dispositivo D-05 sigue conectado al Mac, el controlador podría no cargarse correctamente y, en consecuencia, no ser posible usar el dispositivo D-05.En tal caso, desconecte el dispositivo D-05 y conéctelo a otro puerto USB o encienda el dispositivo D-05 después de iniciar su Mac.
  • Debe desconectar el D-05 del ordenador antes de realizar una actualización de software para macOS. Si ha actualizado el software de macOS con el D-05 conectado, tendrá que instalar de nuevo el controlador.
  • Si el dispositivo D-05 no funciona correctamente después de desactivar el modo Reposo o después de reiniciar el sistema operativo, debería salir de todas las aplicaciones que utilicen D-05 y, a continuación, desconectar y volver a conectar el cable USB de D-05.

Instalación

*Asegúrese de cerrar todas las aplicaciones en ejecución antes. El equipo se reiniciará durante el procedimiento de instalación del driver.
  1. Inicie el ordenador con todos los cables USB desconectados.(excepto el del teclado y el ratón)
    *No conecte D-05 al equipo hasta que se le indique.
  2. Haga doble clic en D05_USBDriver11.pkg para iniciar el instalador.
  3. Siga las instrucciones de la pantalla para continuar con la instalación.
    *Si el mensaje ""Instalador" quiere acceder a los archivos de tu carpeta Descargas." aparece, durante la instalación, haga clic en [Permitir].
  4. Cuando haya terminado la instalación, haga clic en [Reiniciar] para reiniciar el ordenador.
    *Si el sistema le pide que confirme si desea mover el instalador a la Papelera, haga clic en [Conservar].
    *El reinicio del ordenador puede llevar algún tiempo.

Cuando el ordenador se reinicie, conecte el dispositivo D-05 al ordenador con un cable USB y encienda el dispositivo D-05.

Y así concluye la instalación del controlador D-05.

*El archivo Readme y el desinstalador del controlador se instalan en la carpeta Aplicaciones - Roland - D-05 Driver.

Antes de empezar

Configuración para la entrada y salida de audio en macOS

  1. Utilice un cable USB para conectar el D-05 al ordenador y encienda el D-05.
  2. Abra "Preferencias del Sistema" y haga clic en [Sonido].
  3. Seleccione "D-05" para cada uno de los siguientes ajustes:
    • "Reproducir efectos a través de" en la ficha "Efectos de sonido"
    • "Selecciona un dispositivo para la salida de sonido" en la ficha "Salida"
    • "Selecciona un dispositivo para la entrada de sonido" en la ficha "Entrada"

Configuración para la entrada y salida de MIDI en macOS

  1. Utilice un cable USB para conectar el D-05 al ordenador y encienda el D-05.
  2. En la carpeta Aplicaciones - Utilidades, haga doble clic en "Configuración de Audio MIDI" para iniciarlo.
  3. En el menú Ventana, elija "Mostar estudio MIDI" para abrir la ventana Estudio MIDI.
  4. Asegúrese de que aparezca "D-05" en el cuadro de diálogo "Estudio MIDI".
    *Si no aparece "D-05" o aparece en gris, significa que D-05 no se ha reconocido correctamente. Pruebe haciendo clic en "Re-explorar MIDI". Como alternativa, apague el D-05 y vuelva a encenderlo.
    *Si ha instalado un controlador de versión diferente, es posible que la configuración anterior siga activa. Si es así, haga clic en "D-05", que aparece en gris, para seleccionarlo, y luego en "Eliminar dispositivo" para eliminar la configuración anterior.
  5. Para comprobar que la transmisión/recepción MIDI se produce correctamente, siga este procedimiento.
  6. Haga clic en "Probar configuración".
    *En el siguiente paso, el generador de sonido producirá un sonido relativamente alto, así pues es aconsejable bajar el volumen del generador antes de continuar.
  7. Haga clic en la ▼ del D-05 que se muestra en el paso 4. Si se oye un sonido, la configuración se habrá realizado correctamente.
  8. Vuelva a hacer clic en "Probar configuración" para finalizar la prueba.
  9. En el menú Configuración de Audio MIDI, elija "Salir de Configuración de Audio MIDI" para salir de la configuración.

Antes de utilizar el D-05 con software

  1. Utilice un cable USB para conectar el D-05 al ordenador. Seguidamente, inicie la aplicación de software que vaya a utilizar.
  2. Seleccione "D-05" en la configuración de controlador de audio de la aplicación de software.
  3. Seleccione "D-05" en la configuración de controlador de MIDI de la aplicación de software.

Para obtener más detalles, consulte el manual que se incluye con el software.

Para cambiar la configuración (D-05 Driver Setup)

Para cambiar o comprobar la configuración

  1. Utilice un cable USB para conectar el D-05 al ordenador y encienda el D-05.
  2. En la carpeta Aplicaciones - Roland - D-05 Driver, haga doble clic en "D-05 Driver Setup" para iniciarlo.
    Se muestra el panel Configuración del controlador.

Acerca de los elementos del panel Configuración del controlador

"Tamaño del búfer de audio"
Puede ajustar el tamaño de búfer de entrada/salida de audio.
Al reducir el tamaño del búfer se reducirá la latencia y se mejorará el rendimiento en tiempo real, pero podría ser más probable que se oyeran ruidos, dependiendo de la potencia de proceso del equipo y de la carga de datos de canciones.
Al aumentar el tamaño del búfer se alargará la latencia, pero la transmisión de sonido será más estable, y será menos probable que se produzcan ruidos.
En general, es probable que desee reducir el tamaño del búfer para un mejor rendimiento en tiempo real si está tocando un sintetizador de software en tiempo real, o si está monitorizando una entrada de audio que se esté transmitiendo mediante el software DAW.
Al contrario, puede aumentar el tamaño de búfer para evitar los ruidos si está grabando o reproduciendo datos de canciones que contengan varias pistas y con una carga de proceso elevada.
Al cambiar el tamaño del búfer según su situación, puede conseguir el equilibrio correcto entre rendimiento en tiempo real y estabilidad de la transmisión de audio.
 
"Margen de grabación"
Si los datos de audio grabados contienen pérdidas de sonido o ruido incluso después de ajustar el tamaño del búfer en el software y también de ajustar el "Tamaño del búfer de audio," anterior, pruebe con un valor más alto para el "Margen de grabación".
*En algunas aplicaciones de software, quizás no se pueda ajustar el tamaño del búfer.
 
"Sincronización de grabación"
Si la sincronización de los datos de audio grabados no es correcta, ajuste este valor.
 
Nota)
Si hace clic en [Aplicar] mientras se ejecuta el software de audio, los cambios en la configuración no se reflejarán inmediatamente. La configuración solo se aplicará cuando se salga de todas las aplicaciones de software de audio y se inicie de nuevo todo el software.

Para desinstalar

  1. Inicie el ordenador con todos los cables USB desconectados.
    (excepto el del teclado y el ratón)
  2. Haga doble clic en "Uninstaller".
  3. La pantalla indicará "Este procedimiento desinstala el controlador D-05 de este Mac."; haga clic en [Desinstalar].
    *Si aparece algún otro mensaje, siga las instrucciones del mismo.
  4. La pantalla indicará "¿Está seguro de que desea desinstalar el controlador?"; haga clic en [Aceptar].
  5. Aparecerá un cuadro de diálogo en el que se le pide una contraseña; escríbala y haga clic en [Aceptar].
  6. La pantalla indicará "Proceso de desinstalación terminado"; haga clic en [Reiniciar]. El ordenador se reiniciará.
    *El reinicio del ordenador puede llevar algún tiempo.

Solución de problemas

No se puede instalar/desinstalar el controlador

Siga el procedimiento que se explica a continuación para iniciar el instalador/desinstalador.
  1. Mantenga pulsada la tecla Control y haga clic en el icono del instalador/desinstalador.
  2. En el menú que aparece, seleccione "Abrir".
  3. Cuando el sistema le pida que confirme si desea iniciar la aplicación, haga clic en "Abrir".
Compruebe la configuración de seguridad.
  1. En "Preferencias del Sistema", haga clic en "Seguridad y privacidad".
  2. Haga clic en el icono del candado de la parte inferior izquierda. Cuando aparezca un cuadro de diálogo solicitando la contraseña, introduzca la contraseña del administrador para desbloquear.
  3. En la pestaña General, cambie la opción "Permitir apps descargadas de" a "App Store y desarrolladores identificados".
    *Cuando termine de instalar o de desinstalar el controlador, restablezca la configuración original siguiendo el mismo procedimiento.

D-05 no se puede utilizar ni reconocer

¿Ha entrado el ordenador en modo Reposo mientras el D-05 estaba conectado y encendido?
Si el ordenador ha entrado en modo Reposo mientras el D-05 estaba conectado y encendido, es posible que este no funcione correctamente. En este caso, y tras salir de este modo, cierre todas las aplicaciones que utilicen el D-05y, a continuación, vuelva a encender el D-05.
 
Si inicia o reinicia su Mac mientras el dispositivo D-05 sigue conectado al Mac, el controlador podría no cargarse correctamente y, en consecuencia, no ser posible usar el dispositivo D-05.
En tal caso, desconecte el dispositivo D-05 y conéctelo a otro puerto USB o encienda el dispositivo D-05 después de iniciar su Mac.
 

No se oye ningún sonido; no se puede reproducir ni grabar

¿Es posible que haya entrado en modo Reposo durante el proceso de reproducción o grabación?
Si ha entrado en modo Reposo durante la reproducción o la grabación, ya no será posible reproducir ni grabar desde el D-05. Salga de todas las aplicaciones de reproducción o grabación, apague el D-05 y vuelva a encenderlo.
 
Intente ajustar el tamaño del búfer de audio del controlador.
Si el tamaño del búfer de audio del controlador es demasiado pequeño, el sonido podría no reproducirse o podría resultar imposible disfrutar de una reproducción y grabación normales.
En tales casos, consulte "Para cambiar la configuración" y aumente el tamaño del búfer.
 

Se producen ruidos o alteraciones ocasionales en el sonido

Intente ajustar el tamaño del búfer de audio para el software que esté utilizando.
En primer lugar, intente ajustar el tamaño del búfer de audio para el software que esté utilizando.
Si se pierde el sonido o se siguen escuchando ruidos durante la reproducción o grabación, intente ajustar el "Tamaño del búfer de audio" en el panel Configuración del controlador.
*En algunas aplicaciones de software, quizás no se pueda ajustar el tamaño del búfer.
 
¿Es posible que haya realizado una actualización de software macOS con el dispositivo conectado?
Si ha realizado la actualización de software de macOS con el dispositivo conectado, se podrían producir pérdidas en el sonido. Si este es el caso, vuelva a instalar el controlador.
 
¿Está el D-05 conectado a un concentrador USB?
Pruebe a conectar el D-05 directamente al conector USB del propio ordenador.
 

Se producen ruidos o alteraciones ocasionales del sonido en los datos de audio grabados

Si el audio grabado sigue conteniendo pérdidas de sonido o ruidos, incluso después de haber ajustado el tamaño del búfer según lo descrito en "Se producen ruidos o alteraciones ocasionales en el sonido" (arriba), pruebe a aumentar el valor de "Margen de grabación" en el panel Configuración del controlador.
*En algunas aplicaciones de software, quizás no se pueda ajustar el tamaño del búfer.
