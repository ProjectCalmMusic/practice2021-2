Informationen zu diesem Treiber

Dieser Treiber ist erforderlich, um D-05 auf macOS 11 auszuführen.

Funktionen

  • Kompatibel mit den Apple Standards CoreAudio und CoreMIDI
  • Möglichkeit, die Treiberpuffergröße einzustellen sowie Aufnahme- und Wiedergabelatenz exakt festzulegen

Warnungen und Einschränkungen

  • Bevor Sie Ihren Sequencer- oder andere Musiksoftware starten, müssen Sie das D-05 über ein USB-Kabel mit Ihrem Computer verbinden und das D-05 einschalten.
  • Während der Wiedergabe oder der Aufnahme darf das D-05 nicht ausgeschaltet werden und das USB-Kabel nicht getrennt werden. Dadurch könnte die Software oder das Betriebssystem nicht normal beendet werden.
  • Sie müssen zuerst die Sequencer- oder die entsprechende Software beenden. Anschließend können Sie das D-05 ausschalten oder das USB-Kabel trennen.
  • Wenn Sie Ihren Mac hochfahren oder einen Neustart durchführen, während D-05 noch mit dem Mac verbunden ist, wird der Treiber möglicherweise nicht korrekt geladen, sodass Sie D-05 eventuell nicht nutzen können.Trennen Sie D-05 in einem solchen Fall und schließen Sie es an einen anderen USB-Anschluss an bzw. schalten Sie D-05 erst ein, nachdem der Mac hochgefahren wurde.
  • Sie müssen das D-05 vom Computer trennen, bevor Sie eine Softwareaktualisierung für macOS durchführen. Bleibt das D-05 während der macOS Softwareaktualisierung verbunden, muss der Treiber neu installiert werden.
  • Wenn das D-05 nach der Rückkehr aus dem Ruhezustand-Modus oder nach dem Neustart des Betriebssystems nicht ordnungsgemäß funktioniert, sollten Sie alle Anwendungen beenden, die das D-05 verwenden und das USB-Kabel des D-05 trennen und erneut verbinden.

Installation

*Schließen Sie sämtliche Anwendungen, die gerade ausgeführt werden. Ihr Computer wird während der Treiberinstallierung neu starten.
  1. Trennen Sie alle USB-Kabel, und starten Sie den Computer.(außer für Tastatur und Maus)
    *Verbinden Sie D-05 erst mit Ihrem Computer, wenn Sie die entsprechende Anweisung erhalten.
  2. Doppelklicken Sie auf D05_USBDriver11.pkg, um das Installationsprogramm zu starten.
  3. Befolgen Sie die Anweisungen auf dem Bildschirm, um mit der Installation fortzufahren.
    *Falls während der Installierung die Nachricht "„lnstallationsprogramm" möchte Zugriff auf Dateien in deinem Ordner „Downloads"." angezeigt wird, klicken Sie auf [OK].
  4. Klicken Sie nach abgeschlossener Installation auf [Neustart], um Ihren Computer neu zu starten.
    *Wenn Sie bestätigen sollen, dass das Installationsprogramm in den Papierkorb verschoben werden kann, klicken Sie auf [Behalten].
    *Der Neustart des Computers kann u. U. eine Weile dauern.

Wenn der Computer neu startet, schließen Sie D-05 mit einem USB-Kabel an den Computer an, und schalten Sie D-05 ein.

Damit ist die Installation des D-05-Treibers abgeschlossen.

*Die Readme-Datei und das Deinstallationsprogramm für den Treiber werden im Ordner Programme - Roland - D-05 Driver installiert.

Vorbereitung

Einstellungen für Audioeingang und -ausgang im macOS

  1. Verbinden Sie das D-05 über ein USB-Kabel mit Ihrem Computer, und das D-05 einschalten.
  2. Öffnen Sie "Systemeinstellungen", und klicken Sie auf Ton.
  3. Wählen Sie Sie bei jeder der folgenden Einstellungen "D-05":
    • "Toneffekte abspielen über" in der Registerkarte "Toneffekte"
    • "Wählen ein Gerät für die Tonausgabe" in der Registerkarte "Ausgabe"
    • "Wählen ein Gerät für die Toneingabe" in der Registerkarte "Eingabe"

Einstellungen für MIDI-Eingang und -Ausgang auf macOS

  1. Verbinden Sie das D-05 über ein USB-Kabel mit Ihrem Computer, und das D-05 einschalten.
  2. Doppelklicken Sie im Ordner Programme - Dienstprogramme auf "Audio-MIDI-Setup", um die Anwendung zu starten.
  3. Wählen Sie im Menü Fenster "MIDI-Studio einblenden" aus, um das Fenster MIDI-Studio zu öffnen.
  4. Überprüfen Sie, ob im Dialogfeld "MIDI-Studio" das "D-05" angezeigt wird.
    *Wenn das "D-05" nicht angezeigt wird bzw. wenn es ausgegraut ist, wurde das D-05 nicht korrekt erkannt. Klicken Sie auf "Neu suchen". Sie können das D-05 auch ausschalten und dann wieder einschalten.
    *Wenn Sie einen Treiber einer anderen Version installiert haben, werden die vorherigen Einstellungen u. U. beibehalten. Wenn dies der Fall ist, wählen Sie das ausgegraute "D-05" durch Anklicken aus. Klicken Sie anschließend auf "Entfernen", um die vorherigen Einstellungen zu löschen.
  5. Führen Sie folgende Schritte aus, um sicherzustellen, dass MIDI-Empfang und -Übertragung korrekt erfolgen.
  6. Klicken Sie auf "Konfig. testen".
    *Durch den folgenden Schritt wird ein relativ lauter Ton erzeugt. Daher ist es ratsam, vorher die Lautstärke zu reduzieren.
  7. Klicken Sie auf den ▼ des D-05, der in Schritt 4 gezeigt wurde. Wenn Sie etwas hören, wurden die Einstellungen korrekt vorgenommen.
  8. Klicken Sie erneut auf "Konfig. testen", um den Test abzuschließen.
  9. Wählen Sie im Menü Audio-MIDI-Setup "Audio-MIDI-Setup beenden" aus, um die Einstellungen zu verlassen.

Bevor Sie das D-05 mit Software verwenden

  1. Verbinden Sie das D-05 über ein USB-Kabel mit Ihrem Computer. Starten Sie anschließend die Software, die Sie verwenden möchten.
  2. Wählen Sie "D-05" in den Audio-Treibereinstellungen Ihrer Software.
  3. Wählen Sie "D-05" in den MIDI-Treibereinstellungen Ihrer Software.

Weitere Informationen finden Sie in der Bedienungsanleitung für die verwendete Software.

Die Einstellungen ändern (D-05 Driver Setup)

Die Einstellungen ändern oder überprüfen

  1. Verbinden Sie das D-05 über ein USB-Kabel mit Ihrem Computer, und das D-05 einschalten.
  2. Doppelklicken Sie im Ordner Programme - Roland - D-05 Driver auf "D-05 Driver Setup", um die Anwendung zu starten.
    Das Feld Treibereinstellungen wird angezeigt.

Informationen zu den Einträgen im Treibereinstellungen-Feld

"Audiopuffergröße"
Sie können die Puffergröße des Audioeingangs/-ausgangs anpassen.
Durch eine Reduzierung der Puffergröße wird die Latenz verkürzt. Dies steigert die Echtzeit-Performance, erhöht jedoch auch je nach Rechenleistung des Computers und nach Audiodatenlast die Wahrscheinlichkeit von Klick- und Poppgeräuschen.
Eine Erhöhung der Puffergröße führt zu einer Verlängerung der Latenz, aber die Stabilität des Audio-Streaming nimmt ebenfalls zu, und die Wahrscheinlichkeit von Popp- und Klickgeräuschen nimmt ab.
Im Allgemeinen empfiehlt sich eine Reduzierung der Puffergröße zur Verbesserung der Echtzeit-Performance, wenn Sie eine Synthesizer-Software in Echtzeit wiedergeben oder einen Audioeingang über die DAW-Software überwachen.
Umgekehrt können Sie durch eine Erhöhung der Puffergröße verhindern, dass bei der Aufzeichnung oder Wiedergabe von Audiodaten, die aus zahlreichen Titeln bestehen und eine große Verarbeitungsleistung erfordern, Klick- und Poppgeräusche auftreten.
Indem Sie die Puffergröße je nach Situation individuell einstellen, erhalten Sie die richtige Balance zwischen Echtzeit-Performance und Stabilität des Audio-Streams.
 
"Aufnahmebandbreite"
Treten auch nach der Anpassung der Audiopuffergröße in der Software und der genannten "Audiopuffergröße" bei den aufgenommenen Audiodaten Aussetzer im Tonsignal oder störende Geräusche auf, können Sie die "Aufnahmebandbreite"-Einstellung erhöhen.
*Bei einigen Softwareanwendungen ist eine Einstellung der Puffergröße u. U. jedoch nicht möglich.
 
"Aufnahmetaktung"
Stellen Sie diesen Wert ein, wenn die Taktung der aufgenommenen Audiodaten nicht korrekt ist.
 
Hinweis)
Wenn Sie auf [Übernehmen] klicken, während die Audiosoftware ausgeführt wird, werden diese Einstellungen nicht sofort aktiv. Das geschieht erst, nachdem alle Audioanwendungen beendet und die Software neu gestartet wurde.

Deinstallieren

  1. Trennen Sie alle USB-Kabel, und starten Sie den Computer.
    (außer für Tastatur und Maus)
  2. Doppelklicken Sie auf "Uninstaller."
  3. Auf dem Bildschirm wird die Meldung "Hiermit wird der D-05-Treiber von diesem Mac entfernt" angezeigt. Klicken Sie auf [Deinstallieren].
    *Wird eine andere Meldung angezeigt, folgen Sie den Anweisungen in dieser Meldung.
  4. Auf dem Bildschirm wird die Meldung "Möchten Sie den Treiber wirklich deinstallieren?" angezeigt. Klicken Sie auf [OK].
  5. Sie werden über ein Dialogfenster nach einem Kennwort gefragt; geben Sie es ein und klicken Sie auf [OK].
  6. Auf dem Bildschirm wird die Meldung "Die Deinstallation ist abgeschlossen." angezeigt. Klicken Sie auf [Neustart]. Ihr Computer wird jetzt neu gestartet.
    *Der Neustart des Computers kann u. U. eine Weile dauern.

Fehlerbehebung

Der Treiber kann nicht installiert/deinstalliert werden

Führen Sie das unten erläuterte Verfahren aus, um das Installations-/Deinstallationsprogramm zu starten.
  1. Halten Sie die Ctrl-Taste gedrückt, und klicken Sie auf das Symbol für das Installations-/Deinstallationsprogramm.
  2. Wählen Sie im angezeigten Menü "Öffnen" aus.
  3. Wenn Sie zur Bestätigung des Startens der Anwendung aufgefordert werden, klicken Sie auf "Öffnen".
Überprüfen Sie die Sicherheitseinstellungen.
  1. Klicken Sie unter "Systemeinstellungen" auf "Sicherheit".
  2. Klicken Sie auf das Schlosssymbol unten links. Wenn in einem Dialogfeld zur Eingabe eines Kennworts aufgefordert wird, geben Sie zum Entsperren das Administratorkennwort ein.
  3. Ändern Sie auf der Registerkarte Allgemein die Einstellung "Apps-Download erlauben von" zu "App Store und verifizierte Entwickler".
    *Wenn Sie die Installation oder Deinstallation des Treibers abgeschlossen haben, setzen Sie die Einstellung unter Verwendung des gleichen Verfahrens wieder auf die ursprüngliche Einstellung zurück.

Ihr Computer kann D-05 nicht verwenden oder erkennen

Ist der Computer in einen Ruhezustand-Modus übergegangen, während das D-05 verbunden und eingeschaltet war?
Wenn der Computer in einen Ruhezustand-Modus übergegangen ist, während das D-05 verbunden und eingeschaltet war, funktioniert es u. U. nicht ordnungsgemäß. Nachdem der Computer aus dem Ruhezustand zurückgekehrt ist, beenden Sie alle Softwareanwendungen, die das D-05 verwenden, und schalten Sie das D-05 wieder ein.
 
Wenn Sie Ihren Mac hochfahren oder einen Neustart durchführen, während D-05 noch mit dem Mac verbunden ist, wird der Treiber möglicherweise nicht korrekt geladen, sodass Sie D-05 eventuell nicht nutzen können.
Trennen Sie D-05 in einem solchen Fall und schließen Sie es an einen anderen USB-Anschluss an bzw. schalten Sie D-05 erst ein, nachdem der Mac hochgefahren wurde.
 

Es wird kein Ton ausgegeben, die Wiedergabe oder Aufnahme ist nicht möglich

Kann es sein, dass Sie den Computer während der Wiedergabe oder der Aufnahme in den Ruhezustand-Modus versetzt haben?
Wenn Sie den Computer während der Wiedergabe oder der Aufnahme in den Ruhezustand-Modus versetzt haben, können vom D-05 keine Audiodaten mehr wiedergegeben oder aufgenommen werden. Beenden Sie alle derzeit aktiven Softwareanwendungen zur Wiedergabe oder Aufnahme, schalten Sie das D-05 aus und erneut ein.
 
Ändern Sie die Audiopuffergröße des Treibers.
Wenn der Audiopuffer des Treibers zu klein ist, wird möglicherweise kein Ton ausgegeben, oder eine normale Wiedergabe oder Aufnahme ist nicht möglich.
In diesem Fall finden Sie unter "Die Einstellungen ändern" weitere Informationen, wie Sie die Größe des Puffers erhöhen.
 

Gelegentlich treten Unterbrechungen oder störende Geräusche auf

Ändern Sie die Audiopuffergröße in der verwendeten Software.
Ändern Sie zunächst die Audiopuffergröße in der verwendeten Software.
Treten bei der Wiedergabe oder Aufnahme weiterhin Aussetzer im Tonsignal oder störende Geräusche auf, dann ändern bzw. vergrößern Sie die "Audiopuffergröße" im Feld "Treibereinstellungen".
*Bei einigen Softwareanwendungen ist eine Einstellung der Puffergröße u. U. jedoch nicht möglich.
 
Haben Sie die macOS-Software u. U. aktualisiert, während das Gerät angeschlossen war?
Wenn Sie die macOS-Software aktualisiert haben, während das Gerät angeschlossen war, können Aussetzer im Tonsignal auftreten. Ist dies der Fall, installieren Sie den Treiber erneut.
 
Ist das D-05 mit einem USB-Hub verbunden?
Verbinden Sie das D-05 direkt mit einem USB-Anschluss des Computers.
 

Bei den aufgenommenen Audiodaten treten gelegentlich Unterbrechungen im Tonsignal oder störende Geräusche auf

Treten auch nach der Änderung der Audiopuffergröße, wie oben unter "Gelegentlich treten Unterbrechungen oder störende Geräusche auf" beschrieben, bei den aufgenommenen Audiodaten Aussetzer im Tonsignal oder störende Geräusche auf, können Sie im Feld Treibereinstellungen die "Aufnahmebandbreite"-Einstellung erhöhen.
*Bei einigen Softwareanwendungen ist eine Einstellung der Puffergröße u. U. jedoch nicht möglich.
